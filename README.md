# Regulated AI MLOps Platform

The `platform-v0.5.0` branch contains a checksum-verified source snapshot and a one-time GitHub Actions bootstrap workflow.

After merge, the workflow restores the complete project and runs the full test, lint, security, container, governance, monitoring, and release-evidence pipeline before it commits the final source tree to `main`.

The final repository contains the full code, fixed configuration, 5,000-row synthetic dataset, processed features, champion and challenger model artifacts, batch predictions, governance reports, deployment files, and a self-contained evidence dashboard.
